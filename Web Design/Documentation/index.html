About

Python is an easy to learn, powerful programming language which serves as a beginner friendly introduction to programming - specifically Object-Oriented Programming (OOP). It was created by Guido van Rossum and released in 1991. Python's design allows for easy code readability by using whitespace, making it ideal for projects of any size on numerous platforms such as:

  • Creating web applications. Python frameworks like Flask and Django allow users to create softwares easily and quickly.
  • Machine learning. Stock prediction and picking, spam detection and image recongition can be done today using Python modules like Tensorflow.
  • Browser automation. "Automate the boring stuff with Python" is an online free book to teach users how to automate everyday tasks.
  • Handle big data through analytics. Python provides powerful scientific and numerical computing.
  • Connect to database systems such as MongoDB through a Python driver.

This documentation is a concise introduction to many of Python's most basic yet essential features and would give readers a good idea of Python's language and coding style such as using newlines and indentations instead of semicolons and brackets used in other popular languages.

Getting Started

To begin, you could use an online Python IDE and compiler such as repl.it. Once you are ready, let us write our first Python code and press run.

print("Hello World!")

In other languages where using indentations (whitespaces) in code for better readability, in Python it is a necessity. Since Python does not use brackets, indentations represent a block of code. This is done by pressing either tab or space, but the standard indentation is 4 spaces.

# This is a comment, marked out by hashtags '#' # This will work as intended: >>> if 10 > 1: >>>  print('Ten is really bigger than one.') # This will not work and will produce an error: >>> if 10 > 1: >>> print('Ten is bigger than one, but will not print')

Variables

Variables are used to store data values. There is no requirement to declare variables or their type before using since every variable is an object. It can even change type after being set.

>>> x = 15 # Starts off as a number >>> y = 'Apples' >>> print(x) 15 >>> print(y) Apples >>> x = 'Tasty' # Changed to a String >>> print(x) Tasty

You could also assign value to multiple variables in one single line. You can assign multiple various names to variables, which are all case-sensitive.
For example, myName is not the same as MyName.

>>> variableFruit, variableName, variableHobby = 'Apple', 'Mike', 'Programming' >>> print(variableFruit + variableName + variableHobby) Apple Mike Programming

As seen above, Python allows for simple operators such as adding strings or numbers but mixing operators between numbers and strings are not supported.

>>> one = 1 >>> hello = 'hello' >>> print(one + hello) # This will trigger an error.

Numbers

The Python interpreter supports integers, floats and complex numbers. We will run through the most commonly used integers and floats. Integers, or Int, is a whole number without decimals. A floating number, on the other hand, contains decimals.
You can use Python like a calculator using simple operators like plus (+), minus (-), multiply (*) and divide (/).

>>> width = 20 >>> height = 10 >>> area = width * height >>> print(area) 200 >>> 10 / 5 # Division always returns a float 2.0

You can mix integers and operators and even convert from one type to another.

>>> 8 * 2.45 + 3 22.6 >>> float = float(10) # convert int 10 to a float >>> print(float) 10.0

Strings

Strings are defined either with single or double quotes. You can also assign multiline strings by using triple quotes. Often, the use of double quotes is preferred to allow usage of apostrophes.

>>> string = "You're the best!" >>> print(string) You're the best! >>> print("""Hello, ...I am a multi line string""") Hello, I am a multi line string

Previously, we showed that we cannot use operators mixing variable integers and strings. However, we can use operators to concat (glue) strings and repeat using plus (+) and multiple (*) respectively. This only works on string literals and not variables.

>>> 3 * "haha" + " so funny!" 'hahahahahaha so funny!'

Conditions

Python supports a wide range of conditions derieved from mathematics. They are commonly used in "if statements" and loops. A few examples are equals (==), not equals (!=), less than (<), greater than (>), less than or equal (<=) and more than or equal (>=). The coparisons will return a boolean of either True or False.

>>> x = 2 # single '=' sets the variable >>> print(x == 2) # double '==' is used for comparisons True >>> print(x >= 3) False >>> print(x != 3) True

Moreover, boolean operators such as 'and' and 'or' allows for complex expressions. The following example allows for multiple conditions.

>>> age = 10 >>> name = 'Henry' >>> print(age > 8 and name == 'Henry') True >>> print(age >= 10 or name == 'Shaw') True

If Else

Similar to most languages, Python also has the if statement which is the most commonly known. Simply put, the if statement returns true if all the conditions are met. Python will continue to run through all the conditions until one of them is met, otherwise the else statement will execute if applicable. Key the code in and run it.

x = int(input("Please enter an integer: ")) # Method to ask for integer input if (x == 0):  print('You entered a zero') if (x > 0):  print('You entered a positive number') else:  print('You entered a negative number') --- Please enter an integer: 10 You entered a positive number

Instead of using multiple if statement which will all run sequentially if all the conditions are fulfilled, we can use elif statement which will only run if all the previous statements above returns false. The else condition will be executed last if all statement above returns false.

a = 200 b = 150 if (a > b):  print('a is larger than b') elif (a > b + 10):  print('While it is true, it will not print because the first statement is true') --- a is larger than b

While Loops

While loops repeat as long as the condition provided remains true. To prevent infinite loops, we often use statements like break to stop the loop even if the initial condition is true. The continue statement can skip the current block to proceed on to the next iteration of the loop.

i = 0 # The while loop is true when i < 50 but stops early once i > 10 while i < 50:  if (i % 2 == 0): # check for even numbers   print(i, 'is an even number')  i+=1  if (i > 10):   break --- 0 is an even number 2 is an even number 4 is an even number 6 is an even number 8 is an even number 10 is an even number

Unlike other popular languages, Python allows the use of else statement within loops. When the conditions provided fails then the code block in else statement will execute. However, else block is skipped only if break is executed.

i = 0 while(i < 4):  print(i)  i+=1 else:  print('the value has reached 4 and will stop') --- 0 1 2 3 the value has reached 4 and will stop

For Loops

A For loop is used to iterate over a sequence (a list or a string), in the order they appear in the sequence. Within the for loop, we can continue to execute a set of statements until the loop ends.

names = ["John", "Henry", "Jeremy", "Tom"] for name in names:  if name == "Henry":   continue # skip when name is Henry  print(name) --- John Jeremy Tom

We can also use the range function to iterate over a sequence of numbers. The range function is zero based (starts from zero if only one argument is provided). If two arguments are provided, the function is inclusive of the first and exclusive of second argument respectively.

# Prints out numbers 0, 1, 2 for x in range(3):  print(x) --- # Prints out 3, 4, 5 for x in range(3, 6):  print(x)

Functions

A function is a block of code which will only run when it is called. You can pass perameters into a function and also make it return data. We use function to divide our code into useful blocks and bringing order to the code. Benefits include better readability and ability to reuse functions in different part of the code.

Functions in Python are declared using the def keyword, followed by the function's name and can also receive arguments (which are optional). To return data, use the return keyword.

def myFirstFunction():  print('Hello World!') def sum(a, b): # Here we have two parameters  return a + b --- >>> # Call myFirstFunction >>> myFirstFunction() Hello World! >>> print('1 + 2 equals', sum(1, 2)) 1 + 2 equals 3

Functions are also able to utilise a default parameter value if the function is called without a parameter.

def askName(name = 'Stranger'):  print('Hello,', name) --- >>> askName('John') Hello, John >>> askName() Hello, Stranger

Lists

There are four main types of collections (arrays) in Python. Each of them has different properties and it is important to choose the right type. We will touch on three of them in detail. To begin, a list is a collection which is ordered and changeable.

>>> fruits = ['orange', 'apple', 'pear', 'banana'] >>> fruits.append('watermelon') # adds to the back of the list >>> fruits ['orange', 'apple', 'pear', 'banana', 'watermelon'] >>> fruits.index('banana') 3 # Lists have index starting from zero >>> fruits.reverse() >>> fruits ['watermelon', 'banana', 'pear', 'apple', 'orange'] >>> fruits.pop(1) # if no value given, the last item is removed 'banana' >>> fruits ['watermelon', 'pear', 'apple', 'orange']

Moreover, we can obtain the value of items by referring to the index number. Similar to arrays, we can obtain the length of the list and loop through it.

>>> fruits = ['orange', 'apple', 'pear', 'banana'] >>> print(len(fruits)) 4 >>> for fruit in fruits: ...  print(fruit) orange apple pear banana

Tuplets

A tuple consists of a number of values separated by commas. They are enclosed by parentheses and are immutable unlike lists. Similar to lists, they can contain duplicate items and are ordered. Since values in a tuple cannot be changed, any modification actions to the tuple will produce an error.

>>> fruits = ('orange', 'apple', 'pear', 'banana') >>> fruits[2] 'pear' >>> if "pear" in fruits: ...  print("pear is in the tuple") pear is in the tuple >>> fruits[2] = 'watermelon' TypeError: 'tuple' object does not support item assignment

Sets

A set is a collection which is both unordered and inindexed and is written using curly brackets. Unlike lists and tuplets, they do not contain duplicate entries while supporting advanced mathematical operations like union. Once a set is created, the items cannot be changed but functions like add or remove are supported.

>>> fruits = {'orange', 'apple', 'pear', 'banana', 'apple', 'pear'} >>> fruits # duplicates removed {'orange', 'banana', 'pear', 'apple'} >>> fruits.add('watermelon') >>> fruits {'banana', 'watermelon', 'orange', 'pear', 'apple'}

Advanced mathematical symbols are also supported as shown. A few of them are shown below.

>>> a = set('867283729') >>> b = set('06195628193') >>> a {'3', '7', '2', '9', '8', '6'} >>> a - b # number in a but not b {'7'} >>> a & b # number in a and b {'3', '2', '9', '8', '6'}
